Data Analysis of Local Businesses

This project involves analyzing data from various local businesses to identify trends and insights that can drive better decision-making. Using tools such as Python and libraries like Pandas and Matplotlib, I collected, processed, and visualized data on sales, customer demographics, and market conditions. The findings provide valuable information for business owners, helping them understand customer preferences and optimize their strategies. This project not only showcases my analytical skills but also my ability to communicate complex data in an understandable format.

Book Inventory Management System

This project showcases a Python console-based application designed to manage a personal book collection. Using object-oriented principles, the system allows users to add, remove, and search for books while tracking their reading status. The project emphasizes user experience and data management, demonstrating skills in software development and programming logic. With a clear user interface and organized code structure, this system highlights the importance of effective inventory management in everyday life.
